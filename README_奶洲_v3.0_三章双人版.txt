《奶洲：三章双人撤离行动》v3.0
================================

一、启动方式
------------
1. 双击 naizhou_v3_modern_city_duo.html。
2. 建议使用最新版 Chrome、Edge、Firefox 或 Safari。
3. 游戏是单文件离线版：角色图片、界面、怪物图像、宝物图标、音效和三张地图都已内置，不需要安装，也不需要联网。

二、旧内容保留与存档
------------------
本版本在 v2.1 · Lv.20 的基础上追加内容，没有删除第一章、第二章、原有四名奶聪、技能、绝技、怪物、宝物、难度、委托、结算或升级。

永久升级仍为同一个“奶站改造”界面：
- 奶泡护盾：上限 Lv.20
- 扩容奶包：上限 Lv.20
- 奶滴调压：上限 Lv.20

单人和双人共用同一份奶票、同一套永久升级、同一个累计背包和同一本怪物图鉴，不会出现两套升级界面。

游戏继续使用原存档键 naizhou_save_v1。为了尽量继承旧版进度，可将新版文件放在旧文件所在目录，并改成旧文件名后用同一浏览器打开。不同浏览器对本地 HTML 存档隔离方式略有差异。

三、单人 / 双人模式
-----------------
主菜单新增“游玩模式”：
- 单人行动：保留原来的完整玩法与操作。
- 双人同屏：两名奶聪独立拥有生命、护甲、弹药、医疗剂、主动技能和绝技；共享团队背包、撤离收益与永久成长。

双人角色规则：
- 1P 和 2P 可分别选择四种奶聪形态。
- 两人也可以同时选择同一种奶聪。
- 双人模式下，背包容量为两名角色基础容量与永久扩容的团队总和。
- 双人模式会提高怪物生命与伤害，避免人数增加后失去挑战。
- 两人距离过远会受到同屏牵引提示，以保证同一画面可正常协作。
- 一名奶聪生命归零后会进入 18 秒倒地状态；另一名奶聪靠近并按住交互键 2.6 秒即可救援。
- 两人同时倒地，或倒地倒计时结束，行动失败。
- 双人撤离时，两名奶聪都必须存活并同时进入撤离圈。

四、键盘操作
------------
1P：
- WASD：移动
- 鼠标：瞄准
- 鼠标左键 / 空格：开火
- E：搜索、查看容器、救援、撤离
- R：换弹
- Q：医疗
- Shift：主动技能
- F：角色绝技

2P：
- 方向键：移动
- 小键盘 0 / 小键盘 Enter：开火
- 小键盘 1 / 小键盘小数点：搜索、查看容器、救援、撤离
- 小键盘 2：换弹
- 小键盘 3：医疗
- 小键盘 4：主动技能
- 小键盘 5：角色绝技

提示：2P 使用数字小键盘时，请确认 Num Lock 已开启。部分紧凑型键盘没有独立数字小键盘，可继续使用单人模式。

五、累计背包
------------
奶站菜单新增“累计背包”。只有成功撤离的回收物才会永久进入仓库。

累计背包会记录：
- 历次成功带出的总件数
- 所有物品累计估值
- 已发现回收物种类
- 传说收藏数量
- 每种回收物的累计数量
- 每种回收物的累计价值
- 每种回收物的最佳单件价值
- 名称、稀有度、所属章节、基础价值、重量和介绍

可以按第一章、第二章、第三章、传说稀有度和“仅已发现”筛选，也可以搜索名称与描述。

三章共收录 47 种回收物：
- 第一章：12 种经典宝物
- 第二章：16 种魔鬼洞穴回收物
- 第三章：19 种现代城市回收物

六、怪物图鉴
------------
奶站菜单新增“怪物图鉴”，共收录 30 类怪物。

每条图鉴包含：
- 独立绘制的怪物图片
- 怪物名称
- 普通 / 精英 / 首领分类
- 基础生命值
- 所属地图
- 背景介绍
- 主要攻击特点
- 本地累计击倒次数

图鉴可按三章、精英怪、首领筛选，也可搜索名称、介绍或攻击方式。

七、第三章：现代奶都
------------------
风险级别：超高危都市。
整体难度略高于第二章“魔鬼洞穴”：敌人数量更多，基础生命与伤害更高，远程攻击、锁定攻击和区域控制更密集，地图机关会与怪物攻击叠加。

主要区域：
- 奶智地铁总站
- 乳云数据中心
- 冷链金融街
- 奶潮购物中心
- 自动驾驶中枢
- 摩天乳塔
- 霓虹城市主干道与高架路口

城市危险机制：
- 周期电网：先预警，随后对范围内目标造成电击与技能干扰。
- 失控车流：沿道路高速横扫，逼迫玩家改变站位。
- 激光门：短暂锁定后封锁通道。
- 漏电水面：进入后持续受伤。
- 算法广告区：造成方向反转和视觉干扰。
- EMP 禁能场：短时间无法开火技能或绝技。
- 站台牵引场：把奶聪拖向危险中心。
- 城市反盗协议：部分高价值容器会触发伏击或 EMP。

撤离点：
- 西城网约奶车
- 东环磁悬浮门
- 高架检修梯
- 乳塔消防电梯

八、第三章普通怪
----------------
1. 扫码奶尸（基础生命 92）
   被认证系统吞噬的通勤者。使用三连二维码弹、认证扑击和短暂控制干扰。

2. 外卖奶骑（基础生命 118）
   与无人电动车融为一体。会蓄力高速冲锋，并留下持续减速与伤害的滑乳轨迹。

3. 共享单车乳怪（基础生命 104）
   由共享单车堆积形成。擅长侧向移动、回旋轮片和环形车轮弹幕。

4. 监控眼球机（基础生命 96）
   脱离杆体的城市摄像头。使用三连镜头弹，并以预警红线发射远距灼线。

5. 地铁挤压体（基础生命 168）
   由早高峰人群与变质奶泡压缩形成。会释放挤压震波、减速并把玩家拉向自己。

6. 奶云微型机（基础生命 58）
   速度很快的末端执行无人机。使用追踪脉冲和俯冲撞击，常被精英怪批量召唤。
