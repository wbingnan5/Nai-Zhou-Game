三、六名新增奶聪的平衡设计
--------------------------
1. 翠绿奶聪 · 生息型
基础：110 生命 / 44 护甲 / 214 移速 / 18 伤害 / 0.158 秒射击间隔 / 12.5 kg 基础负重
被动「嫩叶代谢」：医疗剂额外恢复 8 生命，并净化灼烧、中毒、寒霜和减速。
Shift「青芽回春」：净化异常，立即小幅恢复；6 秒持续恢复并提高 15% 移速。冷却 24 秒。
F「翠冠森域」：在施法位置建立 7 秒固定领域，治疗范围内队友、缓速并小幅伤害敌人。冷却 46 秒。
平衡点：恢复稳定但爆发低；领域固定，离开后没有收益。

2. 愤怒奶聪 · 狂战型
基础：112 生命 / 38 护甲 / 210 移速 / 22 伤害 / 0.178 秒射击间隔 / 10.5 kg 基础负重
被动「越痛越凶」：生命低于 40% 时伤害 +12%，射击间隔缩短 10%。
Shift「怒火上膛」：5.5 秒伤害 +25%、射速和移速提高，但受到伤害 +18%。冷却 23 秒。
F「暴怒贯拳」：沿瞄准方向突进，窄幅贯穿路径敌人并造成击晕。冷却 42 秒。
平衡点：高风险换爆发，低护甲、低负重、检索慢，主动期间更容易倒地。

3. 爱心奶聪 · 援护型
基础：102 生命 / 48 护甲 / 210 移速 / 17 伤害 / 0.158 秒射击间隔 / 12 kg 基础负重
被动「温柔接力」：使用医疗剂后，为附近队友恢复 12 生命和 4 护甲。
Shift「心心相印」：治疗自身并援护最近队友；单人时获得额外护甲。冷却 25 秒。
F「心动结界」：7 秒范围光环，光环内队友减伤 24% 并持续恢复。冷却 45 秒。
平衡点：十名角色中基础输出最低之一，保护能力不能直接清怪。

4. 吃脚奶聪 · 翻滚型
基础：94 生命 / 42 护甲 / 242 移速 / 19.5 伤害 / 0.17 秒射击间隔 / 10.5 kg 基础负重
被动「离谱身法」：翻滚期间获得 30% 闪避率。
Shift「抱脚翻滚」：4.5 秒移速 +30% 并启用闪避。冷却 20 秒。
F「脚气回旋」：4.8 秒移动贴身回旋，持续伤害并轻微击退周围怪物。冷却 40 秒。
平衡点：身板和负重偏低，绝技必须贴身，危险区域仍然能限制路线。

5. 比耶奶聪 · 节奏型
基础：104 生命 / 44 护甲 / 220 移速 / 18 伤害 / 0.15 秒射击间隔 / 11.5 kg 基础负重
被动「好耶节拍」：每第 6 发普通奶泡弹造成 1.35 倍伤害。
Shift「胜利手势」：立即装弹，获得 4 发 1.65 倍伤害、可穿透一次的胜利弹。冷却 22 秒。
F「全场比耶」：全队 7 秒伤害 +10%、射速 +12%，并立即装弹。冷却 44 秒。
平衡点：增益平稳但没有高额单次处决、隐身、治疗或重装减伤。

6. 崩住奶聪 · 忍耐型
基础：122 生命 / 58 护甲 / 192 移速 / 19.5 伤害 / 0.172 秒射击间隔 / 12.5 kg 基础负重
被动「面不改色」：停止移动时受到伤害降低 10%。
Shift「崩住护体」：3.2 秒受到伤害降低 55%，期间速度略降；技能结束将储存的压力转为冲击波。冷却 24 秒。
F「真的崩不住」：恢复 30 护甲、震开周围怪物，并获得 5 秒 28% 减伤。冷却 45 秒。
平衡点：比爆笑奶聪更依赖站位和时机，移速、检索与射速均偏慢。

四、双人模式
------------
六名新角色可分别分配给 1P 或 2P，也允许两人选择同一形态。
所有角色继续共享奶票、Lv.40 奶站改造、团队背包、累计仓库与怪物图鉴，不增加第二套升级界面。

五、操作
--------
1P：WASD 移动，鼠标/空格开火，E 搜索/撤离，R 换弹，Q 医疗，Shift 主动技能，F 绝技。
2P：方向键移动，小键盘 0/Enter 开火，小键盘 1/小数点搜索撤离，小键盘 2 换弹，小键盘 3 医疗，小键盘 4 技能，小键盘 5 绝技。
